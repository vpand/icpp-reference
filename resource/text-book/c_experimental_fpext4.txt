   Floating-point extensions for C - Part 4: Supplementary functions,
   ISO/IEC TS 18661-4:2015, defines the following new components for the C
   standard library, as recommended by ISO/IEC/IEEE 60559:2011 (the
   current revision of IEEE-754).

   Supplemenatary mathematical functions listed below are merged into C2x
   standard.

Predefined feature test macros

   __STDC_IEC_60559_FUNCS__
   integer constant of type long and value 201506L
   (macro constant)

Notes

   All functions added to the C library by this extension are only
   declared if a macro __STDC_WANT_IEC_60559_FUNCS_EXT__ is defined before
   math.h is included.

   The decimal floating-point variants of every function are only defined
   if __STDC_WANT_IEC_60559_DFP_EXT__ is also defined before math.h is
   included.

   The extended precision variants of every function are only defined if
   __STDC_WANT_IEC_60559_TYPES_EXT__ is defined before math.h is included.

   The correctly-rounded versions of all functions (with the cr- prefix)
   are optional.
