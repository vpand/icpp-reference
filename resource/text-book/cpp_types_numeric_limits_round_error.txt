   static T round_error() throw();
                                             (until C++11)
   static constexpr T round_error() noexcept;
                                             (since C++11)

   Returns the largest possible rounding error in ULPs (units in the last
   place) as defined by ISO 10967, which can vary from 0.5 (rounding to
   the nearest digit) to 1.0 (rounding to zero or to infinity). It is only
   meaningful if std::numeric_limits<T>::is_integer == false.
