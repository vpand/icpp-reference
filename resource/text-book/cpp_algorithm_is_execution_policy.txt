   Defined in header <execution>
   template< class T >
   struct is_execution_policy;
                                (since C++17)

   Checks whether T is a standard or implementation-defined execution
   policy type.

   Provides the member constant value which is equal to true, if T is a
   standard execution policy type, or an implementation-defined execution
   policy type. Otherwise, value is equal to false.

   If the program adds specializations for std::is_execution_policy or
   std::is_execution_policy_v, the behavior is undefined.
