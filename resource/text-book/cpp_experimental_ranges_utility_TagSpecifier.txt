   Defined in header <experimental/ranges/utility>
   template< class T >
   concept bool TagSpecifier = /* implementation-defined */;
                                                  (ranges TS)

   The concept TagSpecifier<T> is satisfied if and only if T is a tag
   specifier for use with ranges::tagged.

   The Ranges TS provides a set of tag specifiers in
   <experimental/ranges/algorithm>. Users cannot define their own tag
   specifiers.
