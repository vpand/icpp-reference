                                                                (ranges TS)

   The class template projected bundles a Readable type I and a function
   Proj into a new Readable type whose reference type is the result of
   applying Proj to the reference type of I. It exists solely to ease
   constraint specification, and so its operator*() may not be actually
   defined.
