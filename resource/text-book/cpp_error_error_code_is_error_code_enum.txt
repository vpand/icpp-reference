   Defined in header <system_error>
   template< class T >
   struct is_error_code_enum;
                                   (since C++11)

   If T is an error code enumeration (such as std::io_errc and
   std::future_errc), this template provides the member constant value
   equal true. For any other type, value is false.

   This template may be specialized for a program-defined type to indicate
   that the type is eligible for std::error_code implicit conversion.
