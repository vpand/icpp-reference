   Compares a std::experimental::function with a null pointer. Empty
   functions (that is, functions without a callable target) compare equal,
   non-empty functions compare non-equal.

   The != operator is synthesized from operator==.
   (library fundamentals TS v3)

Parameters

   f - std::experimental::function to compare

   1,2) !f
   3,4) (bool) f
