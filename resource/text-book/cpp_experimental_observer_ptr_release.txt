   constexpr element_type* release() noexcept;
                                              (library fundamentals TS v2)

   Stop watching the watched object, if any. get() returns nullptr after
   the call.

Parameters

   (none)

Return value

   A pointer to the previously watched object, or nullptr if there was no
   watched object, i.e. the value which would be returned by get() before
   the call.
