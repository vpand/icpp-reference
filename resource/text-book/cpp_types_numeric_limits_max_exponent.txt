   The value of std::numeric_limits<T>::max_exponent is the largest
   positive number n such that \(\scriptsize r^{n-1}\)rn-1
   , where r is std::numeric_limits<T>::radix, is a representable finite
   value of the floating-point type T.

Example

   Demonstrates the relationships of max_exponent, max_exponent10, and
   max() for the type float:
   Run this code
#include <iostream>
#include <limits>

int main()
{
    std::cout << "max() = " << std::numeric_limits<float>::max() << '\n'
              << "max_exponent10 = " << std::numeric_limits<float>::max_exponent
10 << '\n'
              << std::hexfloat << '\n'
              << "max() = " << std::numeric_limits<float>::max() << '\n'
              << "max_exponent = " << std::numeric_limits<float>::max_exponent <
< '\n';
}

   Output:
max() = 3.40282e+38
max_exponent10 = 38

max() = 0x1.fffffep+127
max_exponent = 128
