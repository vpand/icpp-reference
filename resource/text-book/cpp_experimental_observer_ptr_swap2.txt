   template< class W >
   void swap( observer_ptr<W>& lhs, observer_ptr<W>& rhs ) noexcept;
                      (library fundamentals TS v2)

   Specializes the swap algorithm for std::experimental::observer_ptr.
   Swaps the pointers of lhs and rhs. Calls lhs.swap(rhs).

Parameters

   lhs, rhs - observer_ptrs whose contents to swap

Return value

   (none)

Complexity

   Constant.
