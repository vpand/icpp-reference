   template< bool Const >
   class /*sentinel*/;
   (since C++23)
   (exposition only*)

   The return type of zip_transform_view::end when the underlying view is
   not a common_range.

   The type /*sentinel*/<true> or /*sentinel*/<false> treats the
   underlying view as const-qualified or non-const-qualified respectively.
