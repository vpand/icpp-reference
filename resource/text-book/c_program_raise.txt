   Defined in header <signal.h>
   int raise( int sig );

   Sends signal sig to the program. The signal handler, specified using
   signal(), is invoked.

   If the user-defined signal handling strategy is not set using signal()
   yet, it is implementation-defined whether the signal will be ignored or
   default handler will be invoked.

Parameters

   sig - the signal to be sent. It can be an implementation-defined value
   or one of the following values:
   SIGABRTSIGFPESIGILLSIGINTSIGSEGVSIGTERM
   defines signal types
   (macro constant)

Return value

   0 upon success, non-zero value on failure.

Example

   Run this code
#include <signal.h>
#include <stdio.h>

void signal_handler(int signal)
{
    printf("Received signal %d\n", signal);
}

int main(void)
{
    // Install a signal handler.
    signal(SIGTERM, signal_handler);

    printf("Sending signal %d\n", SIGTERM);
    raise(SIGTERM);
    printf("Exit main()\n");
}

   Output:
Sending signal 15
Received signal 15
Exit main()
