   Defined in header <stdatomic.h>
   #define ATOMIC_FLAG_INIT /* unspecified */
                                             (since C11)

   Expands to an initializer that can be used to initialize atomic_flag
   type to the clear state. The value of an atomic_flag that is not
   initialized using this macro is indeterminate.

Example

#include <stdatomic.h>

atomic_flag flag = ATOMIC_FLAG_INIT;
