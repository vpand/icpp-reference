   template< bool Const >
   class /*iterator*/
   (since C++20)
   (exposition only*)

   The return type of join_view::begin, and of join_view::end when both
   the outer range V and the inner range ranges::range_reference_t<V>
   satisfy common_range and the parent join_view is a forward_range.

   If V is not a simple view (e.g. if ranges::iterator_t<const V> is
   invalid or different from ranges::iterator_t<V>), Const is true for
   iterators returned from the const overloads, and false otherwise. If V
   is a simple view, Const is true if and only if
   ranges::range_reference_t<V> is a reference.
