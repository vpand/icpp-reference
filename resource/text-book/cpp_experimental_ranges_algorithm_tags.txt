                                                    (ranges TS)

   These classes are TagSpecifiers for use with ranges::tagged.

   The name of a specifier is also the element name it corresponds to. For
   instance, ranges::tagged<std::pair<int, int>, tag::min, tag::max>
   provides a set of named accessors named min and a set of named
   accessors named max.
