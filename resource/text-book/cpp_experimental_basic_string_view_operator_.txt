   (library fundamentals TS)

   Replaces the view with that of view.

Parameters

   *this

Example

   Run this code
#include <iostream>
#include <experimental/string_view>
int main()
{
    std::experimental::string_view v = "Hello, world";
    v = v.substr(7);
    std::cout << v << '\n';
}
