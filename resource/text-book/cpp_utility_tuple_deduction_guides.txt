   tuple<UTypes...>;
   (5)                                      (since C++17)

   These deduction guides are provided for std::tuple to account for the
   edge cases missed by the implicit deduction guides, in particular,
   non-copyable arguments and array to pointer conversion.

Example

   Run this code
#include <tuple>
int main()
{
    int a[2], b[3], c[4];
    std::tuple t1{a, b, c}; // explicit deduction guide is used in this case
}
