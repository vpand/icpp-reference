   support for type-erased allocators.

Member types removed in Library Fundamental TS v3

   argument_type T if sizeof...(Args) == 1 and T is the first and only
   type in Args...
   first_argument_type T1 if sizeof...(Args) == 2 and T1 is the first of
   the two types in Args...
   second_argument_type T2 if sizeof...(Args) == 2 and T2 is the second of
   the two types in Args...

   Member types argument_type, first_argument_type, and
   second_argument_type are removed, because the correponding member types
   of std::function are removed in C++20.
   (library fundamentals TS v3)
