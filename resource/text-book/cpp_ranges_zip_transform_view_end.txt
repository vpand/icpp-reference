   (2)                       (since C++23)

   Returns an iterator or a sentinel that compares equal to the end
   iterator of the zip_transform_view.

   Let zip_ denote the underlying tuple of views:
   1) Equivalent to:
   if constexpr (ranges::common_range</*InnerView*/>)

       return /*iterator*/<false>(*this, zip_.end());
   else
       return /*sentinel*/<false>(zip_.end());
   2) Equivalent to:
   if constexpr (ranges::common_range<const /*InnerView*/>)

       return /*iterator*/<true>(*this, zip_.end());
   else
       return /*sentinel*/<true>(zip_.end());
